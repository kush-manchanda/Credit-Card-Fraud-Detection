Credit Card Fraud Detection

Built an end-to-end Machine Learning classification project to identify fraudulent credit card transactions by following the complete ML lifecycle, from data acquisition and preprocessing to model development and performance evaluation.

The project was developed using a real-world credit card transactions dataset obtained from Kaggle. It involved data cleaning, preprocessing, and preparing the dataset for effective model training. The primary goal was to accurately classify transactions as fraudulent or genuine using supervised machine learning techniques.

To solve this binary classification problem, a Logistic Regression model was implemented and trained on the processed dataset. The model was evaluated on unseen test data and achieved an Accuracy Score of 96.88%, demonstrating its effectiveness in detecting fraudulent transactions with high reliability.

The complete implementation was carried out in Python using Google Colab, with libraries such as NumPy, Pandas, Matplotlib, and Scikit-learn for data manipulation, visualization, model building, and evaluation. The full project, including the source code and implementation details, has been published on GitHub.
